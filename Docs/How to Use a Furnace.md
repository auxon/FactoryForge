# How to Use a Furnace

Click on the furnace — Opens the Machine UI.

Select a recipe — Click a smelting recipe button at the bottom (e.g., "Iron Plate").

Add ore — Click an input slot (left side):

Transfers 1 matching item from your inventory (e.g., Iron Ore).

You need the selected recipe's input in your inventory.
The furnace smelts automatically:
When it has ore and the recipe is set, it starts smelting.

A progress bar shows completion.

Take output — Click an output slot (right side) to transfer smelted items (e.g., Iron Plates) to your inventory.

Quick Tips:

For Iron Plates:
Select "Iron Plate" recipe → Click input slot to add Iron Ore → Wait for smelting → Click output slot to take Iron Plates

Furnaces need fuel (for Stone Furnaces) — Add Coal or Wood to the fuel slot if needed

The furnace shows available smelting recipes at the bottom when you click it

Try clicking your furnace — the Machine UI should open and you can start smelting.
